# Лабораторная работа 1: Конвертер цветовых моделей

## Описание проекта

Интерактивное приложение для преобразования цветов между тремя цветовыми моделями:
- **RGB** (Red, Green, Blue)
- **CMYK** (Cyan, Magenta, Yellow, Key)
- **HLS** (Hue, Lightness, Saturation)

Приложение предоставляет графический интерфейс с ползунками и текстовыми полями для управления значениями цветов в каждой модели с возможностью выбора цвета из палитры.

## Функциональность

### Основные возможности:
- **Визуальное отображение** текущего цвета
- **Три панели управления** для каждой цветовой модели
- **Слайдеры и текстовые поля** для точного ввода значений
- **Автоматическая синхронизация** - изменение в одной модели мгновенно отражается в других
- **Выбор цвета из палитры** с помощью системного диалога

### Диапазоны значений:

| Модель | Компоненты | Диапазон |
|--------|------------|----------|
| RGB | Red, Green, Blue | 0-255 |
| CMYK | Cyan, Magenta, Yellow, Key | 0-100% |
| HLS | Hue, Lightness, Saturation | 0-360°, 0-100%, 0-100% |

## Формулы преобразования цветов

### Преобразование RGB → CMYK:
```
K = min(1 - R/255, 1 - G/255, 1 - B/255);
C = (1 - R/255 - K) / (1 - K);
M = (1 - G/255 - K) / (1 - K);
Y = (1 - B/255 - K) / (1 - K);
```
**Где:**
- R, G, B ∈ [0, 255]
- C, M, Y, K ∈ [0, 1] (в программе умножаются на 100 для процентного представления)

### Преобразование CMYK → RGB:
```
R = 255 * (1 - C) * (1 - K)
G = 255 * (1 - M) * (1 - K)
B = 255 * (1 - Y) * (1 - K)
```
**Где:**
- C, M, Y, K ∈ [0, 1] (в программе делятся на 100)
- R, G, B ∈ [0, 255]
## Формулы преобразования HLS → RGB

### Основные вычисления:
```
M2 = L + S - L × S если L < 0.5
M2 = L × (1 + S) если L ≥ 0.5
M1 = 2 × L - M2
```

### Алгоритм преобразования:
```
Если S = 0:
R = L × 255
G = L × 255
B = L × 255
Иначе:
R = Value(H + 120, M1, M2) × 255
G = Value(H, M1, M2) × 255
B = Value(H - 120, M1, M2) × 255
```

### Вспомогательная функция Value:
```
Function Value(H, M1, M2):
Если H > 360: H = H - 360
Если H < 0: H = H + 360
Если H < 60: return M1 + (M2 - M1) × H / 60
Если H < 180: return M2
Если H < 240: return M1 + (M2 - M1) × (240 - H) / 60
Иначе: return M1
```

**Где:**
- H ∈ [0, 360] - оттенок (Hue)
- L ∈ [0, 1] - яркость (Lightness)
- S ∈ [0, 1] - насыщенность (Saturation)
- R, G, B ∈ [0, 255] - компоненты RGB
